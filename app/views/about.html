To get it running, you need:

  1. Install yeoman or simply just bower and grunt.
  2. For Sass using You will need to install Ruby.
  3. Install Sass. Ruby uses Gems to manage its various packages of code like Sass. In your open terminal window type: gem install sass.
  4. Once they have been installed go to the directory and install the grunt-cli -> 'npm install -g grunt-cli' and then 'npm install'.
  5. Next run 'bower install' to install all the dependencies required.
  6. Next run 'gulp js' to generate main app.js script.
  7. Once everything has been installed simply run 'grunt serve'.(if you will have any wornings use --force)
